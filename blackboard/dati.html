Confiscati bene - RAW data

Dataset nativo

Il dataset nativo è pubblicato in CC BY 3.0 IT sul sito del Comune di Palermo, in formato .csv.

Dataset rielaborato

Il dataset nativo non si prestava nativamente ad essere utilizzato per l'elaborazione fatta in queste pagine. È stato quindi rielaborato con uno degli strumenti utilizzati: OpenRefine. Allegate anche alcune note.

Dataset convertito in geojson

Per gli elementi del dataset a cui è stato possibile associare in automatico una coppia di coordinate, è stato creato un file in formato geojson, un formato nativo per molte applicazioni di webmapping.
